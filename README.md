Auto BGM Changer v0.3.3
ゲーム画面をリアルタイムで画像判定し、状況に合わせたBGMへ自動的に切り替えるPythonツールです。

主な機能
自動BGM切り替え: 指定した判定範囲を監視し、シーンに合わせたBGMを再生。
Auto Volume Sync: 特定アプリの音量に同期してBGMを自動調音（ダッキング/ブースト）。
音割れ予報UI: ブースト時の最大出力をリアルタイム算出し、音割れリスクを警告。
スマート・フェード: 判定猶予とフェード時間を個別に設定し、滑らかな切り替えを実現。
低負荷設計: 監視間隔を最大10秒まで設定可能。バックグラウンド動作に最適化。


推奨動作環境
OS: Windows 10 / 11 (64bit)
CPU: 4コア以上推奨（画像判定処理をバックグラウンドで行うため）
メモリ: 8GB以上推奨（BGMのキャッシュ機能により、楽曲数に応じてメモリを消費します）
ディスプレイ: シングル / マルチディスプレイ対応


動作仕様・注意点

対応画面モード
本ツールはWindowsのデスクトップ複製APIを使用して画面をキャプチャします。以下のモードでの動作を推奨します。

推奨: ウィンドウモード、または 仮想フルスクリーン（ボーダーレスウィンドウ）
非推奨: 専用フルスクリーン（Exclusive Fullscreen）
※専用フルスクリーンモードでは、ゲームがデスクトップ管理をバイパスするため、画面の判定が停止したり、真っ暗になったりする場合があります。

楽曲ファイルについて
本ツールにはBGMファイルは付属していません。
ご自身で用意したファイルを bgm/ フォルダに配置して使用してください。


セットアップ

必要ライブラリ
pip install pygame opencv-python numpy bettercam psutil pycaw sv-ttk darkdetect


フォルダ構成
実行ファイルと同じ階層に以下のフォルダを作成してください。

bgm/: BGMファイル（scene1.mp3, default.mp3 等）
target/: 判定用画像（scene1.png 等）
amb/: (オプション) 環境音用

使い方
python main.py を実行します。
判定したい画面に合わせて「判定範囲」や「厳しさ」を調整します。
「開始」ボタンで自動監視とBGM再生が始まります。
