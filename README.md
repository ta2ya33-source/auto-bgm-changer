Auto BGM Changer v0.3.3

ゲーム画面の状況をリアルタイムで画像判定し、状況に合わせたBGMへ自動的に切り替えるPythonツールです。


主な機能

自動BGM切り替え: 指定した判定範囲の画像（テンプレート）を監視し、シーンに合わせたBGMを再生します。
Auto Volume Sync: 特定のアプリケーション（EXE）の音量レベルに同期して、BGMを自動で「ダッキング（下げる）」または「ブースト（上げる）」します。
音割れ予報UI: BGM音量設定時に、ブーストがかかった際の「想定最大出力」をリアルタイムで算出し、音割れ（クリッピング）のリスクを警告します。
スマート・フェード: シーン切り替え時の「判定猶予」と「フェードアウト時間」を個別に設定可能。滑らかな音響演出を実現します。
低負荷設計: 監視間隔を最大10秒まで拡張可能。バックグラウンドでの動作負荷を最小限に抑えます。


セットアップ

必要ライブラリ

以下のライブラリが必要です。
pip install pygame opencv-python numpy bettercam psutil pycaw sv-ttk darkdetect


フォルダ構成

実行ファイルと同じ階層に以下のフォルダを作成し、ファイルを配置してください。
bgm/: 再生したいBGMファイル（scene1.mp3, default.mp3 など）
target/: 判定に使用する画像（scene1.png など）
amb/: (オプション) 環境音用フォルダ


使い方

python auto\_bgm\_changer.py を実行します。
判定したいゲーム画面に合わせて「判定範囲」や「厳しさ」を調整します。
「開始」ボタンを押すと、自動監視とBGM再生が始まります。
